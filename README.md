# אזרחות 2026 - הכנה לבגרות באזרחות

## סקירת הפרויקט
- **שם**: Civics2026 (אזרחות 2026)

3. Service Worker לתמיכה offline
4. Cache headers לנכסים סטטיים
5. Lazy loading לקבצי נתונים גדולים (476K deferred)

### Phase 3 - תכונות חסרות ✅

5. שינוי סיסמת מורה

### Phase 4 - Backend מלא (Cloudflare D1) ✅
1. בסיס נתונים D1 עם 6 טבלאות
2. CRUD API לנתוני תלמידים
3. דשבורד מורה עם ניהול כיתתי
4. דוחות ואנליטיקה

### Phase 5 - השלמת הפרויקט ✅
1. מיפוי 96 שאלות בגרות ל-16 יחידות (90/96 ממופים)

7. 18/18 בדיקות E2E עברו בהצלחה

## ארכיטקטורת נתונים

### טבלאות D1
| טבלה | תיאור |
|-------|--------|
| `students` | נתוני תלמידים (id, שם, כיתה) |
| `progress` | התקדמות לפי יחידה (checklist, תשובות, מצב רוח) |
| `notes` | הערות אישיות לשאלות |
| `highlights` | הדגשות טקסט |
| `sync_log` | יומן סנכרון |
| `analytics` | אירועי אנליטיקה |

### נקודות קצה API
| Method | Path | תיאור |
|--------|------|--------|
| GET | `/api/health` | בדיקת תקינות |
| POST | `/api/sync` | שמירת נתוני תלמיד ב-D1 |
| GET | `/api/sync/:studentId` | שליפת נתוני תלמיד מ-D1 |
| GET | `/api/teacher/students?password=` | רשימת תלמידים (דורש סיסמה) |
| POST | `/api/teacher/change-password` | שינוי סיסמת מורה |
| POST | `/api/analytics` | שמירת אירוע אנליטיקה |
| GET | `/api/analytics/summary?password=` | סיכום אנליטיקה |
| DELETE | `/api/student/:id?password=` | מחיקת תלמיד |
| GET | `/api/student/:id/export` | ייצוא מלא של תלמיד |

## מדריך למשתמש

### תלמידים
1. הזינו שם וכיתה בפינה העליונה
2. בחרו יחידת לימוד מהתפריט
3. למדו מושגים → קראו חומר מיקוד → תרגלו שאלות → הקליטו תשובה בעל-פה
4. עקבו אחר ההתקדמות בדף הבית

### מורים
1. לחצו על "דשבורד מורה" בתפריט
2. הזינו סיסמה (ברירת מחדל: `civics2026!`)
3. צפו בהתקדמות תלמידים, צרו מתווה שיעור, ייצאו נתונים

## פריסה

### Platform
- **Cloudflare Pages** עם Hono backend
- **D1 SQLite** לאחסון נתונים
- **Service Worker** לתמיכה offline

### פקודות
```bash
npm run build             # בניית הפרויקט (כולל minification)
npm run dev:d1            # פיתוח מקומי עם D1
npm run db:migrate:local  # יישום migrations מקומי
npm run deploy            # פריסה ל-Cloudflare Pages
```
